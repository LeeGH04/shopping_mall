# shopping_mall

## 프로젝트 개요
이 프로젝트는 **AI(인공지능) 기술을 사용하지 않고** 순수한 웹 개발 기술을 활용하여 개발되었습니다.  
쇼핑몰 웹사이트의 프론트엔드와 백엔드를 구현하며, 학습과 협업 능력을 향상시키는 것을 목표로 합니다.

---

## 사용한 기술 및 프레임워크
### **프론트엔드**
- **React**: 사용자 인터페이스 구축
- **React Router**: 클라이언트 사이드 라우팅
- **Styled-Components / TailwindCSS**: 스타일링
- **Axios**: API 연동

## 주요 기능
- 상품 목록 조회
- 상품 상세 정보 보기
- 장바구니에 상품 추가/삭제
- 사용자 계정 관리 (로그인/회원가입)
